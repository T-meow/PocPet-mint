# Pocpet-Mint

## 玩法

- 照顾宠物的饱腹、清洁、心情、体力与健康，获取经验、小心心并提升等级。
- 在商店购买食物、礼物和护理用品，也可以批量使用背包中的普通道具。
- 使用番茄钟完成专注与休息循环，推进每日愿望、回归任务和离线事件。
- 经营花园，解锁地块、种树、浇水、施肥和升级工具，收获道具、金币与金苹果。
- 安排伙伴日程，培养技能、结识 Mod 邻居并领取每日奖励。
- 使用金币或扭蛋券参与金苹果扭蛋，收集增益卡、成就、奖杯并完成伙伴梦想。
- 生日、相遇纪念日、节日和月初会提供日期奖励，错过后可在限定时间内补领。
- 支持 Mod v1/v2、自定义安全道具，以及存档导入、导出、备份和损坏恢复。
