# 项目记录

- 项目名：`PocPet`
- 作者：FrostForge Studio

## 项目概述

PocPet 是 FrostForge Studio 制作的一款可爱的挂机休闲游戏，核心体验围绕陪伴、照顾和轻量成长展开。玩家通过喂食、清洁、玩耍、睡眠、工作和购物等短操作照顾宠物，在在线与离线过程中持续积累金币、爱心、心情变化和日常事件。

游戏节奏偏轻松，不强调高压数值对抗，而是通过稳定的挂机收益、即时反馈、随机小事件、商店道具、食物偏好、天气影响和宠物状态变化，形成适合碎片时间游玩的陪伴感。PocPet 的目标体验是让玩家每天打开后都能看到宠物的细微变化，并通过少量操作获得明确反馈。

当前项目支持 Windows 独立运行版和 Android 测试包，前端使用 Web 技术实现，桌面和移动端通过 Tauri 打包。资源体系包含宠物状态图、活动图、商品图标、背景音乐、UI 音效、动作音效和宠物反馈音效。后续方向包括更丰富的待机场景、梦境事件、冒险派遣、长期收集、亲密等级和更多装饰/礼物内容。
## 打包规则

- 当前版本：1.0.0，版本来源为 package.json；Tauri 和 Cargo 配置同步为同一版本。

- Windows 只保留带版本号的独立运行版：`release/PocPet-<version>-windows-portable.exe`。
- Android 只保留带版本号的测试用包：`release/PocPet-<version>-arm64-release-debugsigned.apk`。
- Windows 打包命令：`npm.cmd run package:win:portable`。
- Android 打包命令：`npm.cmd run package:android:arm64`。
- Android 常规 Tauri 打包在当前 Windows 环境会卡在/失败于 `.so` 符号链接创建，当前脚本会复用或生成 arm64 `.so`，复制到 Android `jniLibs`，再执行 Gradle、zipalign 和 debug 签名。
- `release` 目录最终只保留当前版本交付物，例如 `PocPet-1.0.0-windows-portable.exe` 和 `PocPet-1.0.0-arm64-release-debugsigned.apk`。
- Android 应用图标已改为与 Windows app 同源，来源为 `src-tauri/icons/icon.png`。
- Android 底部系统导航栏遮挡问题：已移除 `enableEdgeToEdge()`，前端使用 `safe-area-inset-bottom` 给底部操作栏和页面留出安全区。

## 当前玩法和资源

- 宠物当前是单形态视觉表现，状态图包括：`pet_idle_sit.png`、`pet_hungry.png`、`pet_dirty_sad.png`、`pet_little_dirty.png`、`pet_reading_books.png`、`pet_sick1.png`、`pet_sleep.png`。
- 活动表现图包括：玩耍 `pet_happy.png`、清洁 `pet_bath.png`、工作 `pet_work_making_food.png` / `pet_work_watering_plants.png`、吃饼干 `pet_eat_cookie.png`、吃便当 `pet_eat_noodles.png`、营养餐/烧烤猪蹄 `pet_eat_meat.png`、用药/爱心反馈 `pet_give_heart.png`、升级 `pet_levelUp.png`。
- 商店图标已补齐当前商品映射，包括应急饼干、便当、营养餐、烧烤猪蹄、草莓蛋糕、AD 钙奶、小花束、闪亮贴纸、软云玩偶、铃铛丝带等。
- 当前经济和状态规则：初始金币 30、爱心 0；工作产出金币；饼干每日限量；存在偏好食物、每日折扣、天气心情、睡眠结算、梦话、梦境事件、离线事件、过度操作反馈、低体力限制等系统。
- 音频素材已按用途整理到 `src/assets/audio`：`bgm`、`ui`、`action`、`pet`。部分音频仍是合并素材，例如 `action_work_play_medicine.mp3`、`pet_hunger_sad_sick.mp3`、`pet_sleep_wake_touch.mp3`。
